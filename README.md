# ClassMate

Monorepo layout with separate frontend and backend.

## Structure

- `frontend/`: Vite + React app
- `backend/`: Express API

## Run (frontend)

```bash
cd frontend
npm install
npm run dev
```

## Run (backend)

```bash
cd backend
npm install
npm run dev
```

## Root scripts (optional)

```bash
npm run dev:frontend
npm run dev:backend
```

## Build (frontend)

```bash
cd frontend
npm run build
npm run preview
```
